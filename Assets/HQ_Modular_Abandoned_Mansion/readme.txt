Make sure Unity setting is set to Linear and Deferred.

Setup (Desktop):

1. Import Standard Asset from Unity Asset Store.

2. Drop in an FPS controller in the scene.

3. Attach PostProcess Layer on the main camera and set Layer to default or everything.

4. Add PostProcess.asset on Post Process Volume Profile. (Located in Assets\HQ_Modular_Abandoned_Mansion\Scenes\Demo_Profiles)

5. Done
